# 长周期推广任务

正式推广任务不是一篇推文的待办，而是一段时间内围绕业务目标持续推进的工作单元。

建议生命周期：

1. 思考记录：老板或团队保存初步构思，可继续修改。
2. 确认立项：使用者明确确认后，进入正式任务。
3. 策略形成：确定目标、人群、主线、产品排布与阶段指标。
4. 内容与节奏：形成内容矩阵、渠道计划和承接动作。
5. 执行反馈：记录发布、用户反馈、销售反馈和转化数据。
6. 阶段复盘：识别有效线索、无效动作和下一轮调整。
7. 迭代或结束：进入下一阶段，或在达成目标后归档。

AI 可以辅助生成策略、标题、内容和节奏，但不得自动完成第 2 步，也不得未经确认修改正式任务状态。
